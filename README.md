# Café Aether ☕

Café Aether is a calm, minimal landing page designed for a cozy neighborhood café.
The goal was to create a warm digital presence that reflects the café’s atmosphere and invites visitors in.

Features:
– Clean and modern layout
– Mobile-responsive design
– Clear call-to-action sections
– Soft color palette inspired by coffee culture

Built with: HTML, CSS, JavaScript
Live Demo: https://myrrinvalea.github.io/
